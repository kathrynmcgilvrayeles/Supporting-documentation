Step 1

Adding content

Select the '+' sign and then add the documents you would like the AI to help you with.

❯

Step 2

Type of docs

You can upload different types of documents and formats into the prompt including:

Text and Document Files

  • .txt - Plain text files
  • .doc / .docx - Microsoft Word documents
  • .pdf - Portable Document Format (great for formatted reports, resumes, etc.)
  • .md - Markdown files

Data Files

  • .csv - Comma-separated values (ideal for spreadsheets and data analysis)
  • .xlsx - Microsoft Excel files
  • .json - JavaScript Object Notation (structured data)
  • .xml - Extensible Markup Language

Image Files

  • .jpg / .jpeg
  • .png
  • .gif
  • .bmp
  • .tiff

Audio Files

  • .mp3 - MPEG Audio Layer III
  • .wav - Waveform Audio File Format
  • .flac - Free Lossless Audio Codec

Video Files

  • .mp4 - MPEG-4 Video File
  • .avi - Audio Video Interleave
  • .mov - Apple QuickTime Movie

Presentation Files

  • .ppt / .pptx - Microsoft PowerPoint presentations

Archive Files

  • .zip - Zipped file
  • .rar - Roshal Archive file

Code Files

  • .py - Python script
  • .js - JavaScript file
  • .html - HyperText Markup Language file
  • .css - Cascading Style Sheets file
❯

Step 3

Webpage

You can copy and paste a website link directly into the AI’s prompt field. The AI will then read and process the content of that webpage, allowing you to ask questions, summarise, or generate new content based on the information it contains.

This is incredibly useful for referencing online articles, news, or specific web pages without manually copying all the text.

❯

Step 4

Snipping tool or Screenshot

Sometimes, the information you need isn’t easily copy-pasted or linked. A snipping tool (or screenshot feature) allows you to capture specific parts of your screen – an image, a table, a diagram, or even text within an image – and upload it as an image file.

The AI can then use its visual recognition and optical character recognition (OCR) capabilities to understand and interact with the content within that image. You don’t need to save the image. You can snip or screenshot and paste it into Copilot.

This is incredibly useful when you have taken a screenshot with a lot of text and you don’t feel like typing. Ask the AI to type the text in the image.
